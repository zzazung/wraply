# Wraply AI Development Rules

1. Do not rename existing variables or functions.
2. Do not remove original code unless explicitly instructed.
3. Always use semicolons in JavaScript.
4. If code changes, provide FULL SOURCE CODE.
5. Always provide tests for new functionality.
6. Do not change project structure without approval.
7. Prefer repository file references over pasted code.
8. Maintain compatibility with existing build pipeline.
